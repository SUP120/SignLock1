## 🚀 How to Run (3 Steps)

### Step 1: Open in Android Studio
```
File → Open → Select SignLock folder
Wait for Gradle sync
```

### Step 2: Connect Your Phone
```
Enable USB Debugging
Connect via USB
Allow debugging prompt
```

### Step 3: Run
```
Click green Run button (▶️)
Or press Shift + F10
```

**That's it!** App will install and launch.

## 🐛 If Something Goes Wrong

### App won't build?
```
File → Invalidate Caches → Restart
```

### Lock screen not appearing?
```
Check permissions are granted
Toggle protection on in home screen
```

### Signature not matching?
```
Draw more naturally
Check speed (not too slow/fast)
```
